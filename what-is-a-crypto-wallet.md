# What is a crypto wallet

## Overview
A crypto wallet is a tool that allow users to store, send and receive crypto currency securley

##Types of Wallets 

### Hot Wallets
- Connected to the internet
- Easy to access
- Example: Metamask, coinbase wallet

### Cold Wallet
- Not connected to the internet
- More secure
- Example: Ledger, trezor

## How it works
1. You create a wallet
2. You get a unique wallet address
3. You use that address to send and receive crypto
